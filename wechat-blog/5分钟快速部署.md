# 🚀 5分钟快速部署指南

> 只需要5分钟，让您的微信公众号文章网站上线！

## 📋 准备工作

✅ **需要准备的东西:**
- GitHub账户 (免费注册: https://github.com)
- 一个简单的文本编辑器
- 5分钟时间

## 🎯 第一步: 创建GitHub仓库 (1分钟)

1. **登录GitHub**
   - 打开 https://github.com
   - 使用您的账户登录

2. **创建新仓库**
   - 点击右上角绿色按钮 **"New"**
   - Repository name: `wechat-blog`
   - Description: `微信公众号文章展示网站`
   - 选择 **Public** (GitHub Pages要求)
   - ✅ 勾选 "Add a README file"
   - 点击 **"Create repository"**

## 📂 第二步: 上传网站文件 (2分钟)

### 方法A: 拖拽上传 (推荐)

1. **在GitHub仓库页面**
   - 点击 **"uploading an existing file"** 链接
   - 或者直接拖拽文件到页面中央

2. **批量上传**
   - 将 `wechat-blog` 文件夹中的所有内容
   - 直接拖拽到上传区域

3. **提交上传**
   - 在页面底部写提交信息: `Initial commit: WeChat blog website`
   - 点击 **"Commit changes"**

### 方法B: 单个文件上传

如果拖拽有问题，可以逐个上传核心文件:
1. **先上传**: `index.html`
2. **上传**: `assets` 文件夹 (整个文件夹)
3. **上传**: `posts` 文件夹 (整个文件夹)  
4. **上传**: `README.md` 等其他文件

## ⚙️ 第三步: 启用GitHub Pages (1分钟)

1. **进入设置**
   - 在仓库页面点击 **"Settings"** 标签

2. **配置Pages**
   - 找到左侧 **"Pages"** 选项
   - Source: 选择 **"Deploy from a branch"**
   - Branch: 选择 **"main"** (或 **"master"**)
   - Folder: 选择 **"/ (root)"**
   - 点击 **"Save"**

## 🌐 第四步: 访问您的网站 (1分钟)

1. **等待部署**
   - 页面会显示: "Your site is ready to be published"
   - 等待1-2分钟让GitHub完成部署

2. **获取网站地址**
   - 设置页面会显示: `https://您的用户名.github.io/wechat-blog`
   - 复制这个链接到浏览器

3. **🎉 完成！**
   - 您的网站已经上线了！

## 🔗 您的网站地址

部署完成后，您的网站将可以通过以下地址访问:

在手机上打开同样的链接，网站会自动适配手机屏幕！

## 🔄 日常更新

### 添加新文章
1. 编辑 `posts` 文件夹中的文章
2. 编辑 `assets/js/main.js` 添加新文章到列表
3. 在GitHub页面:
   - 点击 **"Add file" > "Upload files"**
   - 上传修改后的文件
   - 填写提交信息: `Add new article`
   - 点击 **"Commit changes"**
4. GitHub Pages会自动更新您的网站

### 修改网站样式
- 编辑 `assets/css/style.css`
- 上传修改后的文件
- 提交更改

## 🛠️ 遇到问题？

### 网站不显示？
- ✅ 检查GitHub Pages设置是否正确
- ✅ 确认分支名称 (main vs master)
- ✅ 等待3-5分钟让部署完成

### 图片不显示？
- ✅ 确认图片文件上传到 `assets/images/` 文件夹
- ✅ 检查文件名是否正确

### 需要帮助？
- 查看详细指南: `GitHub_Pages_部署指南.md`
- GitHub Pages官方文档: https://pages.github.com/
