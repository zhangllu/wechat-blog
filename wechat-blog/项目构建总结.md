**部署步骤**:
1. 将 `wechat-blog` 目录上传到托管平台
2. 设置为静态网站
3. 配置自定义域名 (可选)
4. 启用HTTPS (推荐)

### 方式二: 传统Web服务器

**支持平台**:
- Apache + mod_rewrite
- Nginx
- IIS
- 任何支持静态文件的Web服务器

**配置要点**:
1. 确保所有MIME类型正确设置
2. 配置目录索引为 `index.html`
3. 启用Gzip压缩 (可选)
4. 设置适当的缓存策略
