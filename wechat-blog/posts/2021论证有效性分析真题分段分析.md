常言道：“耳听为虚，眼见为实。”其实，“眼所见者未必实”。

从哲学意义上来说，事物的表象不等于事物的真相。我们亲眼看到的，显然只是事物的表象而不是真相。只有将表象加以分析，透过现象看本质，才能看到真相。换言之，我们亲眼看到的未必是真实的东西，即“眼所见者未必实”。

举例来说，人们都看到旭日东升，夕阳西下，也就是说，太阳绕地球转。但是，这只是人们站在地球上看到的表象而已，其实这是地球自转造成的。由此可见，眼所见者未必实。

我国古代哲学家老子早就看到了这一点。他说过，人们只看到了房子的“有”（有形的结构），但人们没看到的“无”（房子中无形的空间）才有实际效用。这也说明眼所见者未必实，未见者为实。

老子还说，讲究表面的礼节是“忠信之薄”的表现。韩非子解释时举例说，父母和子女因为感情深厚而不讲究礼节，可见讲究礼节是感情不深的表现。现在人们把那种客气的行为称作“见外”，也是这个道理。这其实也是一种“眼所见者未必实”的现象。因此，如果你看到有人对你很客气，就认为他对你好，那就错了。

第一段

(1)常言道: “耳听为虚, 眼见为实。” 其实, “眼所见者未必实”。

提出中心论点：眼所见者未必实。

# 大纲参考分析：

1. 核心概念的界定前后不一致，“眼见为实”的“实”和文中“眼所见者未必实”的“实”内涵不同。

# 【参考写法】

首先，“眼见为实”的“实”和文中“眼所见者未必实”的“实”内涵不同。其中，“眼见为实”中的“实”通常指客观存在的情况，而“眼所见者未必实”似乎将“实”理解为“本质”或“事物的本真”，前后概念界定不一致，削弱了论证的逻辑严密性。113字，6行

# 第二段

②从哲学意义上来说，事物的表象不等于事物的真相。③我们亲眼看到的，显然只是事物的表象而不是真相。④只有将表象加以分析，透过现象看本质，才能看到真相。⑤换言之，我们亲眼看到的未必是真实的东西，即“眼所见者未必实”。

论证结构：  $② \rightarrow ③$  ；  $④ \rightarrow ⑤$

# 大纲参考分析：

2. 亲眼看到的，其实不只是事物的表象，也可能是真相。

# 【参考写法1】

首先，材料过于绝对地认为“亲眼看到的只是事物的表象而非真相”，忽视了亲眼所见有时可以反映出事物的真相。例如，日常生活中很多事物本身就是所见即所得的，眼见所见未必总是表象。将表象与真相截然分开，过于极端，不符合生活实际。109字，6行

# 【参考写法2】

首先，表象未必需要加以分析才能看到真相。如果事物的真相也清晰地呈现在了表面，那我们在看到表象的同时，无需分析也能直接看到真相。另外，如果观察者知识全面、经验丰富，认知事物的能力较强，那不需要对表象进行分析也能直接看到真相。111字，6行

# 第三段

⑥举例来说，人们都看到旭日东升，夕阳西下，也就是说，太阳绕地球转。⑦但是，这只是人们站在地球上看到的表象而已，其实这是地球自转造成的。⑧由此可见，眼所见者未必实。

论证结构：  $⑥ + 7\rightarrow 8$

# 大纲参考分析：

3. 地球自转的实情，不能用来否定我们看到的“旭日东升，夕阳西下”这一实况。

# 【参考写法】

其次，文章使用地球自转的例子来论证“眼所见者未必实”，并不恰当。虽然旭日东升、夕阳西下的现象是由地球自转造成的，但从地球上观察到的这一自然现象是真实发生的，并非虚假。这一例子更多地说明了观察角度的不同，而不能否定“眼见”的真实性。115字，6行

# 第四段

⑨我国古代哲学家老子早就看到了这一点。他说过，人们只看到了房子的“有”（有形的结构），但人们没看到的“无”（房子中无形的空间）才有实际效用。⑩这也说明眼所见者未必实，未见者为实。

论证结构：  $⑨ \rightarrow ⑩$

# 大纲参考分析：

4. 房子的“无”具有“实际效用”，但这不是“未见者为实”之“实”(真实)。

# 【参考写法】

再次，材料引用老子的“有”与“无”来论证“眼所见者未必实，未见者为实”，论据和结论不相关。老子的观点强调房屋中“无形的空间”才是其实际效用所在。“实际效用”的“实”，强调的是用途而非真实性。另外，人们只看到了房子的“有形结构”，反而是真实存在的。123字，7行

# 第五段

⑪老子还说，讲究表面的礼节是“忠信之薄”的表现。韩非子解释时举例说，父母和子女因为感情深厚而不讲究礼节，⑫可见讲究礼节是感情不深的表现。⑬现在人们把那种客气的行为称作“见外”，也是这个道理。⑭这其实也是一种“眼所见者未必实”的现象。⑮因此，如果你看到有人对你很客气，就认为他对你好，那就错了。

论证结构：  $11\to 12$  ；  $13\rightarrow 14\rightarrow 15$

# 大纲参考分析：

5. 父母和子女因为感情深厚而不讲究礼节，不能推出讲究礼节是感情不深的表现。  
6. 有人对你很客气，也有可能真的对你好。

# 【参考写法1】

最后，父母和子女因为感情深厚而不讲究礼节，也无法说明“讲究礼节是感情不深的表现”。感情深厚的家庭成员确实可能不拘礼节，但这并不意味着所有讲究礼节的人感情不深。将礼节与感情强度直接对应，忽视了个体间的多样性和具体情境的差异性，结论过于片面。119字，6行

# 【参考写法2】

最后，文章认为“眼见的客气不代表真实的好意”，这一观点显得过于绝对。事实上，客气行为的背后动机可能是多方面的，既可能出于礼貌的惯例，也可能真实反映对他人的关心与善意。将所有客气行为都等同于“见外”，忽视了人际交往中复杂多变的心理和情感动机。120字，6行
