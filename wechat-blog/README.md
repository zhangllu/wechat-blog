# 微信公众号文章静态网站

这是一个现代化的响应式静态网站，用于展示从微信公众号转换而来的文章。

## 🌟 功能特性

### 核心功能
- **📱 响应式设计**: 完美适配桌面端、平板和移动设备
- **🔍 智能搜索**: 实时搜索文章标题和内容
- **🏷️ 分类筛选**: 支持按转换状态筛选文章
- **📖 文章展示**: 美观的文章阅读界面
- **🖼️ 图片支持**: 展示文章中的图片资源
- **⚡ 快速加载**: 优化的性能和用户体验

### 技术特性
- **纯静态网站**: 无需后端服务器，部署简单
- **现代UI设计**: 采用Material Design设计语言
- **优秀的SEO**: 搜索引擎友好的结构
- **无障碍访问**: 支持屏幕阅读器和键盘导航

## 🚀 快速开始

### 本地预览

1. **启动本地服务器**
   ```bash
   # 使用Python 3
   python -m http.server 8080
   
   # 或使用Python 2
   python -m SimpleHTTPServer 8080
   
   # 或使用Node.js
   npx serve .
   ```

2. **访问网站**
   打开浏览器访问: `http://localhost:8080`

### 生产部署

1. **上传文件**
   - 将整个 `wechat-blog` 目录上传到您的Web服务器
   - 确保所有文件路径正确

2. **配置Web服务器**
   - 配置为静态文件服务器
   - 确保支持HTML5 History API (可选)

3. **域名配置**
   - 将域名指向Web服务器
   - 配置SSL证书 (推荐)

## 🎨 界面预览

### 主页功能

- **搜索栏**: 实时搜索文章

- **桌面端**: 网格布局，最多3列文章
- **平板端**: 网格布局，最多2列文章  
- **手机端**: 单列布局，优化的触摸体验

## 🛠️ 自定义配置

### 修改网站信息

编辑 `index.html` 中的以下部分:

```html
<title>您的网站标题</title>
<!-- 以及其他文本内容 -->
```

### 修改主题颜色

编辑 `assets/css/style.css` 中的CSS变量:

```css
:root {
    --primary-color: #07c160;        /* 主色调 */
    --secondary-color: #1989fa;      /* 次要色调 */
    --background: #f7f8fa;           /* 背景色 */
    --surface: #ffffff;              /* 表面色 */
}
```

### 添加新文章

1. 将Markdown文件放入 `posts/` 目录
2. 更新 `assets/js/main.js` 中的文章列表
3. 复制相关图片到 `assets/images/` 目录
